# Começando com o projeto


 ## Dependências Necessárias
    React-Router-Dom@v6
    Axios
    react-redux
    redux-thunk
    reduxjs/toolkit

No repositório você pode inicializar:

### `npm start`

Roda o app em modo de desenvolvimento.
Abra [http://localhost:3000](http://localhost:3000) para ver em seu navegador.
